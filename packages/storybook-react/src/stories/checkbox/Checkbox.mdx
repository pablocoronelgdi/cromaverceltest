# Checkbox

El checkbox es un elemento de interfaz de usuario que permite a los usuarios seleccionar o deseleccionar opciones individuales.
Proporcionando una forma clara y simple de recopilar información binaria.
Su versatilidad lo hace adecuado para selección única o múltiple de opciones.
<br />

<Primary />

## Usos

### Componente Controlado
El input puede ser controlado si se le pasa como prop un valor a "checked" y una funcion para modificar
su valor a la prop "onChange"

<Story of={CheckboxStories.ControlledCheckBox} />
<Source of={CheckboxStories.ControlledCheckBox} />

### Componente Sin Controlar
Tambien puede utilizarse sin pasarle ningun valor. Se puede acceder a tu estado mediante el parámetro
"e.target.checked"

<Story of={CheckboxStories.UnControlledCheckBox} />
<Source of={CheckboxStories.UnControlledCheckBox} />

### Con valor por defecto
Los componentes no controlados pueden tener un valor por defecto mediante la prop "defaultChecked".
<Story of={CheckboxStories.DefaultValue} />
<Source of={CheckboxStories.DefaultValue} />

### Deshabilitado
Mediante la prop "disabled" se puede deshabilitar el componente impidiendo su uso. 
Si es un componente controlado, esto no impedira que cambie su valor si dicho valor puede ser modificado
desde otro componente ajeno al checkbox.

<Story of={CheckboxStories.Disabled} />
<Source of={CheckboxStories.Disabled} />
