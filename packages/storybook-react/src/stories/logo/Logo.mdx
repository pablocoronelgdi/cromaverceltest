# Logo

Mediante el componente Logo se podran elegir entre una variedad de Logos del ecosistema Macro. 
<br />

<Primary />

## Usos

### Default
Mediante las props "$name", "$size" y "$colorPrimary" se puede renderizar el componente y modificar su variante, tamaño y color.
