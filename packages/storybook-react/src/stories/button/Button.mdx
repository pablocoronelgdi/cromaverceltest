# Button

Los botones son componentes que hacen referencia a una acción que la persona
 usuaria tiene que realizar. También son conocidos como Call To Action o CTA´s.
  La finalidad de los botones siempre será istar a la persona a decidir cómo quiere
   continuar su experiencia. Pueden utilizarse para empezar o terminar flujos,
    también pueden contener una o varias acciones. 
<br />

<Primary />

## Usos

### Tipo de botón
Mediante la prop "as" se puede seleccionar el tipo de botón, puede ser un button o un anchor.
Esto es por si se quiere usar el componente para navegación entre páginas internas o hacia alguna sección de la misma página.


### Función
Mediante la prop onClick nativa del componente se puede utilizar para disparar algún evento.

### Variante 
Mediante la prop $variant se define el estilo del botón, las opciones son: fill, outline, ghost, y tonal.

### Tamaño
Mediante la prop $size se define el tamaño del botón, las opciones son: extra-small, small, medium, y large.

### Texto
Mediante la prop $text se define el texto o contenido del botón.

### Ícono
Mediante la prop $iconName se define el nombre del icono que se quiere mostrar en el botón.

### Posición del ícono
Mediante la prop $iconPosition se define la posición del ícono junto al texto, las opciones pueden ser "left" o "right" 

### Color de fondo
Mediante la prop $backgroundType se define el tono del color de fondo donde se utiliza el boton, las opciones pueden ser "light" o "dark"

### Ancho máximo
Mediante la prop $fullWidth de tipo Boolean, se define si el componente ocupa todo el ancho posible o del contenido.

### Deshabilitado
Mediante la prop disabled de tipo Boolean, se define si el componente esta deshabilitado.
