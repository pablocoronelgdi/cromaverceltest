# Pictogram

Mediante el componente Pictogram se podran elegir entre una variedad de Pictogramas del ecosistema Macro. 
<br />

<Primary />

## Usos

### Default
Mediante las props "$name", "$size", "$segmentName", y "$backgroundColor" se puede renderizar el componente.

### Pictogram con $backgroundColor Dark
La prop $backgroundColor define el estilo del color de fondo donde se va a renderizar el Pictograma.
- "Light"
- "Dark"

<Story of={PictogramStories.DocPictogramWithBackgroundColor} />
<Source of={PictogramStories.DocPictogramWithBackgroundColor} />

### Pictogram con individuosColor
La prop $individuosColor define el estilo del color secundario del Pictograma, solo se debe usar en si el mismo se aplica en el segmento "Individuos".
- "Blue"
- "Pink"
