# RadioButton

Un radio input es un elemento de interfaz gráfica que permite al usuario seleccionar una opción exclusiva de entre varias disponibles.

<br />

<Primary />

## Usos

### Default

RadioButton input necesita de un valor booleano en la propiedad "checked" y una funcion en la prop "onChange"
para modificarlo.

<Story of={RadioButtonStories.Default} />
<Source of={RadioButtonStories.Default} />

### Deshabilitado

Mediante la prop "disabled" se puede deshabilitar el componente impidiendo su uso.
