# TabsSegmented
Mediante el componente TabsSegmented se podrá navegar entre secciones según los items que se definan en la prop $items del componente
<br />

<Primary />

## Usos

### Tabs Segmented con 2 ítems (Default)
Mediante la prop $items se puede definir las tabs y el contenido que se quiere navegar. 
La prop $item recibe un Array de objetos, este objeto debe tener el siguiente formato cómo mínimo:

id: string
$label: string
$content: ReactNode
