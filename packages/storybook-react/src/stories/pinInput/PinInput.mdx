# PinInput
Mediante el componente PIN Input se podrá ingresar un código numérico
<br />

<Primary />

## Usos

### Default
Solo con importar el componente se puede renderizar con una longitud ($pinLength) de 4 por default

<Story of={PinInputStories.PinInputDefault} />
<Source of={PinInputStories.PinInputDefault} />

### PIN Input con label
Mediante la propiedad $label se pueden definir un label para el input

<Story of={PinInputStories.PinInputLabel} />
<Source of={PinInputStories.PinInputLabel} />

### PIN Input length
Mediante la propiedad $pinLength se puede definir la longitud del valor del input que se quiere renderizar, por default es 4 y cómo máxmimo es 6

<Story of={PinInputStories.PinInputLength} />
<Source of={PinInputStories.PinInputLength} />

### PIN Input deshabilitado
Mediante la propiedad "disabled" se puede deshabilitar el componente impidiendo su uso

<Story of={PinInputStories.PinInputDisabled} />
<Source of={PinInputStories.PinInputDisabled} />

### PIN Input error
Mediante la propiedad "$error" se define el estado de error del componente

<Story of={PinInputStories.PinInputError} />
<Source of={PinInputStories.PinInputError} />

### PIN Input visibility
Mediante la propiedad "$visibility" con un valor "false", hace que se oculten los digitos del componente, similar a un input de tipo "password"

<Story of={PinInputStories.PinInputVisibility} />
<Source of={PinInputStories.PinInputVisibility} />

### PIN Input controlado
El componente puede ser controlado si se le pasa un valor a la propiedad "value" y una funcion de cambio de estado a la propiedad "$onPinChange"

<Story of={PinInputStories.PinInputControlled} />
<Source of={PinInputStories.PinInputControlled} />

### PIN Input on complete
Mediante la propiedad "$onComplete" se puede recibir una función que se ejecutará cuando todos los casilleros del componente hayan sido completados

<Story of={PinInputStories.PinInputOnComplete} />
<Source of={PinInputStories.PinInputOnComplete} />
